# myDingDong
学习并用 java 重写的一个叮咚买菜小插件。仅适用于安卓系统。

1. 使用方法：

（1）提前在叮咚买菜中将需要的蔬菜添加到购物车。

（2）打开插件，在辅助功能设置中授予权限。

（3）退出插件，打开叮咚买菜，点击购物车。此时插件自动帮助点击。

2. 原理：安卓系统提供的辅助功能。
